# Kenya Forecast Folder

This folder contains forecast-side products for the Kenya production pipeline.

```bash
raw/                      # Raw hindcast and forecast NetCDF files
bias_corrected/           # Generated daily bias-corrected NetCDF files
geo_4cropModel/           # Input point/soil folders used for DSSAT sampling
dssat_handoff/            # Generated DSSAT-ready RDS weather/soil handoff
dssat_weather/            # Optional direct DSSAT weather exports
extremes/                 # Forecast climate extremes
Onset_DoY/                # Rainfall onset products
logs/                     # Run logs
diagnostics/              # Verification outputs
manifests/                # Output manifests
```

For the active one-month lead convention, the September-initialized forecast
will be generated by:

```bash
Rscript main/Forecast/run_forecast_to_dssat.R \
  --country KEN \
  --season-start-month 10 \
  --season-year 2025 \
  --season-length-months 3 \
  --lead-months 1 \
  --n-cores 4
```

`geo_4cropModel/` is input. `dssat_handoff/` is output.
