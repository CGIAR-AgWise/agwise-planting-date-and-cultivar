# Kenya DSSAT Template Use Case

This folder stores DSSAT template files used by the formatter:

```bash
data/usecases/useCase_Kenya_Example/Maize/DSSAT/
```

Generated DSSAT transform outputs are not treated as source data. Older
generated transform outputs were archived under:

```bash
data/archive/legacy_20260528/KEN/pre_one_month_lead_oct_init/dssat_transform/
```

The next production DSSAT formatting run will recreate `transform/` outputs as
needed.
